# business-analytics-dashboard-powerbi
Developed a end-to-end Business Analytics Dashboard using Power BI, enabling data-driven decision-making through interactive visualization and key performance insights.
# Executive Overview
This page provides a high-level summary of overall business performance through key metrics and visual insights. It includes KPIs such as total revenue, total orders, average order value, and average delivery time.A stacked bar chart is used to analyze revenue distribution across different cities. A detailed table highlights the top 10 restaurants based on total revenue, average rating, and order volume.   Additionally, a donut chart represents the breakdown of order status, providing visibility into order completion stages. Time-based analysis is supported through line charts that track revenue trends and order trends over a period.This overview enables quick understanding of business performance and supports data-driven decision-making.
# Advanced Insights
This page provides deeper analytical insights using interactive and dynamic visualizations. It includes a slicer with key metrics such as Average delivery time, Total orders, Average order value, and Total revenue.A dynamic line chart is implemented, which updates based on the selected metric, enabling flexible trend analysis. A scatter plot is used to analyze the relationship between revenue, customer rating, and cost, helping identify performance patterns.Additionally, a Q&A feature is integrated to allow users to ask natural language questions and instantly retrieve insights from the data.This page enhances exploratory data analysis and enables users to gain actionable insights through interactive features.
